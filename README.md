# 🌱 Buy Furniture

A modern and interactive web application built using **HTML, CSS, JavaScript, GSAP, and Locomotive Scroll**. The project is designed to raise awareness about the environmental impact of excessive wooden furniture consumption and encourage users to consider sustainable alternatives.

## 🎯 Project Objective

The primary goal of this project is to promote environmental awareness by highlighting how high demand for wooden furniture contributes to deforestation.

Similar to how awareness-based applications encourage social change, this project aims to educate users about making eco-friendly purchasing decisions and supporting sustainable practices to help protect forests and the environment.

## ✨ Features

* Modern and interactive user interface
* Smooth scrolling experience using Locomotive Scroll
* Engaging animations powered by GSAP
* Awareness-driven content about deforestation
* Responsive design for different screen sizes
* Interactive sections to improve user engagement
* Clean and visually appealing layout

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* GSAP (GreenSock Animation Platform)
* Locomotive Scroll

## 🎨 Animation & User Experience

To create a more engaging experience, the project utilizes:

### GSAP

* Smooth entrance animations
* Scroll-triggered effects
* Interactive transitions
* Enhanced visual storytelling

### Locomotive Scroll

* Smooth scrolling behavior
* Improved navigation experience
* Better user interaction and engagement

The combination of GSAP and Locomotive Scroll helps transform a simple awareness website into an immersive and interactive experience.

## 🌍 Environmental Awareness

Forests play a crucial role in maintaining ecological balance, absorbing carbon dioxide, and supporting biodiversity. Through this project, users are encouraged to:

* Understand the impact of deforestation
* Explore sustainable furniture choices
* Support environmentally responsible products
* Contribute to forest conservation efforts

The website serves as an awareness initiative rather than a commercial furniture marketplace.
